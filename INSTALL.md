# Quick Installation Guide

## Install the Extension in Chrome

1. **Open Chrome Extensions Page**
   - Type `chrome://extensions/` in your address bar
   - Or click ⋮ (menu) → Extensions → Manage Extensions

2. **Enable Developer Mode**
   - Toggle the "Developer mode" switch in the top-right corner

3. **Load the Extension**
   - Click "Load unpacked"
   - Navigate to: `D:\PROJECTS\ShortsAutoScroll`
   - Click "Select Folder"

4. **Verify Installation**
   - You should see "Auto Shorts Scroller & Liker v1.0.0"
   - Extension icon should appear in toolbar
   - If not visible, click the puzzle icon and pin it

## Test the Extension

1. **Navigate to YouTube Shorts**
   - Go to https://www.youtube.com/shorts
   - Or click any Shorts video

2. **Open Extension**
   - Click the extension icon in toolbar
   - Accept disclaimer on first use

3. **Verify Detection**
   - Should show "✅ YouTube Shorts detected"
   - Start button should be enabled

4. **Configure Settings** (Optional)
   - Target Videos: 5 (for testing)
   - Wait Min: 10 seconds
   - Wait Max: 15 seconds

5. **Click Start**
   - Watch the countdown timer
   - Check activity log for actions

6. **Click Stop** (anytime)
   - Stops automation immediately

## Troubleshooting

- **Manifest Errors**: Ensure all files are in the correct location
- **Not Detecting Shorts**: Refresh the YouTube page and reopen popup
- **Start Button Disabled**: Make sure you're on a Shorts page (URL contains `/shorts/`)
